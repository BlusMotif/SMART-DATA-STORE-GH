## Troubleshooting Quick Links

**Issue:** "Cannot find module"
- Solution: Run `npm install --production`

**Issue:** Port already in use
- Solution: Change PORT in .env or kill process using port

**Issue:** Database connection failed
- Solution: Verify DATABASE_URL format and credentials

**Issue:** Frontend not loading
- Solution: Check .htaccess in Apache config, verify dist/public exists

**Issue:** High memory usage
- Solution: Restart server, check for memory leaks in logs

**Issue:** Build failed
- Solution: Check Node version (should be 20.x), run `npm run build`
