## If Something Goes Wrong

### Workflow Failed?
1. Click on the failed workflow in Actions tab
2. Expand the error message
3. Common issues:
   - **"FTP connection failed"** ‚Üí Check credentials are correct
   - **"Cannot find dist/"** ‚Üí Build failed, check build logs
   - **"Permission denied"** ‚Üí FTP user needs write access to public_html

### App Not Loading?
1. Check .env file on Hostinger exists
2. Verify DATABASE_URL is correct
3. Check Node.js is running (PM2 or similar)
4. Check browser console for JS errors

### Database Connection Error?
1. Verify DATABASE_URL in .env on Hostinger
2. Check Supabase firewall allows Hostinger IP
3. Test connection manually via SSH
